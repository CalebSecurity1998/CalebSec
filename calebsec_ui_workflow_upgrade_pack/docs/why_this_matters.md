# Why CalebSec Matters

CalebSec was built to simulate the type of work performed by junior SOC analysts and security operations teams.

The platform demonstrates practical experience with:

- Reviewing security alerts
- Prioritizing incidents by severity
- Mapping suspicious behavior to MITRE ATT&CK
- Enriching indicators of compromise
- Documenting analyst findings
- Creating and tracking cases
- Understanding security operations workflows

Rather than being a standalone script, CalebSec functions as an integrated SOC operations platform. This makes it useful as a cybersecurity portfolio project because it shows both technical development ability and security workflow understanding.
