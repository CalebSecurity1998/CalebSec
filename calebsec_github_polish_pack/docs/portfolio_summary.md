# CalebSec Portfolio Summary

## Project Name

CalebSec Security Operations Platform

## Summary

CalebSec is a modular SOC-style cybersecurity platform designed to simulate real-world security operations workflows. It includes SIEM monitoring, alert triage, IOC enrichment, phishing detection, MITRE ATT&CK mapping, case management, and analyst investigation notes.

## Why I Built It

I built CalebSec to strengthen my practical cybersecurity skills and demonstrate hands-on experience with security monitoring, incident response workflows, threat detection, and defensive security tooling.

## Real-World Relevance

CalebSec mirrors common SOC analyst responsibilities, including reviewing alerts, prioritizing incidents, enriching suspicious indicators, documenting investigation notes, and mapping activity to known attacker techniques.
