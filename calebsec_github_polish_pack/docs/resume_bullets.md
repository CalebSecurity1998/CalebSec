# Resume Bullets for CalebSec

Use one of these depending on the role.

## Main Version

Developed an integrated SOC operations platform featuring SIEM monitoring, alert triage workflows, IOC enrichment, phishing detection, MITRE ATT&CK mapping, and case management using FastAPI and SQLite.

## SOC Analyst Version

Built a SOC-style alert triage platform that simulates analyst workflows including alert prioritization, investigation notes, escalation tracking, IOC review, and MITRE ATT&CK mapping.

## Detection Engineering Version

Designed detection and enrichment workflows for suspicious login activity, phishing indicators, malware beaconing behavior, and suspicious command execution using Python, FastAPI, and SQLite.

## Threat Intelligence Version

Implemented a threat intelligence module capable of analyzing IPs, domains, URLs, hashes, and command indicators with local risk scoring, IOC history tracking, and MITRE ATT&CK mapping.

## GitHub Project Description

CalebSec is a FastAPI-based SOC operations platform that combines SIEM-style monitoring, alert triage, threat intelligence enrichment, phishing detection, case management, and MITRE ATT&CK mapping.
